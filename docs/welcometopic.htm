Welcome to the wonderful world of BeOS programming using BRIE !

BeOS Rapid Integrated Environment (or BRIE for short) is an IDE for rapid development of BeOS / OBOS / Zeta applications.

All code is generated in C/C++ using the BeOS API plus a few extensions, templates, etc.
